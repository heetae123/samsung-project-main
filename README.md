# RMS System Marketing Site

SEO-first Next.js App Router site for the RMS System hospitality platform.

## Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- TypeScript

## Commands

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`

## Deployment

After local verification, this project can be connected to Firebase App Hosting as a standard Next.js repository.
